## 주요 내용
**프로젝트 구성 및 개발**
- 5개의 서비스에 대해 Docker 기반 컨테이너 인프라 및 네트워크 구축
- Easy-to-build를 테마로, docker-compose를 통해 명령어 한 줄로 배포할 수 있도록 구성
- Python의 FastAPI 프레임워크를 통한 RESTful 백엔드 구성
- Nginx 기반 프록시 서버 구축 및 MySQL 스키마 설계 및 DAO 구현
- 오픈소스(Docker-py, Code-server) API 활용에 따른 생산성 향상
