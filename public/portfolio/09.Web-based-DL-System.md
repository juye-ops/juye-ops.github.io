## 주요 내용
**오픈소스 리버스 엔지니어링**
- Vue JS 및 Flask 기반의 COCO Annotator 리버스 엔지니어링을 통한 코드 이해
- 오픈소스 커스터마이징을 통한 개발 시간 대폭 감소

**프로젝트 개발 및 서빙**
- On-Premise 서버 운용 및 관리
- Docker(compose)를 통한 인프라 관리
- REST API 및 Socket 기반 서비스 별 데이터 입출력 구현
- Socket을 통한 학습 및 추론 진행도 가시화
- 워크숍 논문 작성
