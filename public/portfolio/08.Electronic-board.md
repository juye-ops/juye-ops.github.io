## 주요 내용
**프로젝트 개발 및 서빙**
- 온프레미스 기반 베어메탈 인프라 구축
- 사용자 정보를 입력할 Kiosk 디바이스 관리 및 Flask 기반 로그인 기능 구현
- REST API 및 Socket 기반 서비스 별 데이터 입출력 구현
- MongoDB 관리 및 운용
- Proxy를 배치하여 클라이언트가 직접 백엔드를 통해 데이터베이스를 접근하는 현상 해결
- 학부, 성별, 학번 등 사용자의 특징을 바탕으로 키워드 정리 및 Word cloud 생성
