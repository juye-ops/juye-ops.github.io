## 주요 내용

**VKS 플랫폼 구성**
- Supervisor 배포 및 관리
- ClusterAPI를 통한 클러스터 커스터마이징
- 고객사 VKS 마이그레이션 대비 아키텍처 구성
- 고객사 플랫폼 구축 제안서 및 운영자 매뉴얼 작성

**Private AI 배포**
- DSM 을 통해 RAG 용 엔터프라이즈 데이터베이스 구성
- KnowledgeBase를 위한 Object Storage 구성
- VCF Automation을 통해 Private AI 구성 및 데이터베이스 연동

**네트워크 제어**
- 네트워크 라이센스가 없는 제약적인 환경에서 외부 컴포넌트 및 인프라를 통해 환경 구성
- HAProxy 기반 클러스터 LB, VyOS 기반 L3 제어 / 네트워크 컴포넌트(DNS, DHCP 등) 구성

**플랫폼 내부 속도 개선**
- ESXi의 Virtual CPU 개념을 조정하여 vCenter가 1분에 10초 가량 통신을 잃던 문제 해결
