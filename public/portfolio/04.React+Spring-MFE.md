## 주요 내용
**프레임워크 별 마이크로 서비스 개발**
- React Module Federation을 통한 Micro-Frontend 구성
- Spring을 통해 멀티 모듈 Micro-Backend 구성
- Swiper 를 통한 애니메이션 구현

**마이크로 서비스 DevOps 엔지니어링**
- Helm Chart 구성 및 ArgoCD/Github ARC 기반 CI/CD 구성
- App-of-apps 아키텍처를 통한 프로젝트 가시성 확보
- Loki/Promtail를 통한 로그 전송 및 Grafana를 통한 모니터링 제공
