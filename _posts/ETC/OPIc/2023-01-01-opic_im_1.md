# OPIc IM Series 1
If we graduated element/middle/high school(even university), we are better at English than we think.

Never remember any scripts.

### **Question**
I would like to know where you live.
- Describe your home.
- What does it look like?
- How many rooms does it have?
- Give me a description with lots of details.


## 1. Never ask 'Ava' a question
If you must, ask a rhetorical(*require No-Answer*) question.

### Example
```
Oh, You wanna know about my place? (X)

Oh, You wanna know about my place.. alright, ~ (O)
```

## 2. Do not use difficult words that you are uncomfortable with.
use simple words you are totally confident in using!

### Example
```
I live in a 4-storey house. (X)

My home is really small. (O)
```

### home vs house vs apartment
- home: 거주지(whatever you live house or apartment)
- house: 주택
- apartment: 아파트

## 3. Try not to repeat the same word in the same sentence

### Example
```
My place is a very small place. (X)

It`s a very small place (O)
My place is very small (O)
```

## 4. Don`t use two descriptive or feeling words using "and"!
If you do, you will sound like a script.
Separate one-sentence into two-sentences
```
My room is comfy and cozy(X)

My room is just so, you know, comfy.
I mean, it`s just so cozy.
```
